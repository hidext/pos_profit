Margin display on POS-Point of Sales Back-end With Margin Analysis Odoo apps

If you are using a POS and want to calculate Margin on each product and want to see the total Margin, then this will be the good Odoo apps for use. Using this odoo module you will able to see the margin product line wise as well as it will display the total Margin while placing an POS Order. You can track the Margins not only on the POS screen, but in database also.

Features

Product Margin for POS Orders

User can see details of product margin of every ordered product in back-end pos order.

Total Product Margin

User can see details total margin of ordered product in back-end pos order.

Details of Product Margin by Category

User can see details of product margin by category.

Product Margin Report for All POS Orders

User can see product margin report for all pos orders in back-end.
